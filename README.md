# Forensics Analysis

Presentation for the **Forensics 101** workshop.

Built with [Slidev](https://sli.dev).

## Content

- Disclaimer & TLP:RED
- Environment setup
- Digital Forensics fundamentals
- Data Acquisition (disc image, mobile, KAPE)
- Artifact Analysis (defanging, Diamond Model, ACH)
- Memory Analysis (Volatility)
- Real-world examples
- Forensics Challenge (Windows 2008 R2 incident)
- Summary & Resources

## Usage

### GitHub Codespaces

Open this repository in GitHub Codespaces — the environment will be configured automatically and the presentation will open in the browser.

### Local

```bash
npm install
npm run dev
```

The presentation will be available at [http://localhost:3030](http://localhost:3030).

### Build

```bash
npm run build
```

Output is placed in the `dist/` directory.
