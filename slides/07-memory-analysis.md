# Memory Analysis — Key Plugins

| Category | Plugin |
|----------|--------|
| **Processes** | `(windows\|linux\|mac).pslist`, `pstree`, `psxview` |
| **Connections** | `(windows\|linux\|mac).netstat`, `netscan` |
| **Commands (CLI)** | `(windows\|linux\|mac).cmdline`, `cmdscan` |
| **File Dump** | `(windows\|linux\|mac).pslist --pid XXXX --dump` |
| **Network traffic** | Via Network Miner Pro (Cap Loader) |
